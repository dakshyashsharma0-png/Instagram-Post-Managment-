# Instagram-Post-Managment-
Instagram History Management System (Linked List in C++)

A menu-driven C++ application that simulates an Instagram user history management system using a Singly Linked List Data Structure. This project demonstrates dynamic memory allocation, node insertion/deletion, searching, and traversal operations in Data Structures using C++.

Features
Insert user details:
At beginning
At end
At a specific position
Delete user details:
From beginning
From end
From a specific position
Search user captions by username
Display all Instagram history records
Count total user nodes
Menu-driven console interface
Technologies Used
C++
Object-Oriented Programming (OOP)
Singly Linked List
Dynamic Memory Allocation
Concepts Covered
Classes & Objects
Constructors
Pointers
Linked Lists
Traversal Operations
Insertion & Deletion Algorithms
Searching Techniques
Sample User Data

Caption

Doubly Linked List Instagram Management System
